# Expense Tracker

A budget tracking web application designed to help users manage, categorize, and visualize their spending. It includes a form for adding new expenses, a dashboard with summaries and charts, data storage, and a contact form.

## Summary of Features
- Add new expenses with detailed information, including description, amount, category, and date  
- Dashboard for visual summaries and expense tracking  
- Contact form to send messages via email  
- Dynamic charts using Chart.js to visualize spending by category  
- Automatic calculation of total monthly spending and largest expense category  
- Multi-page layout:  
  - `index.html` — Home page & monthly summary  
  - `add_expense.html` — Expense form with fields for description, amount, category, and date  
  - `dashboard.html` — Includes bar chart, table, total spent, highest spending category, expense count, and option to delete expenses  
  - `contact.html` — Contact form to send messages via email
